## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Python 3.10.6

### Installation

1. Clone the repo

```sh
   git clone
```

2. Install Python packages

```sh
    pip install -r requirements.txt
```

3. Run the app
